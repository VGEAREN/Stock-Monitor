# 如何制作 GitHub 风格的带背景虚化截图

## 推荐工具

### 1. shots.so（免费，最常用）

1. 截取应用窗口：`Cmd + Shift + 4`，按住 `Space`，点击窗口（自动带阴影）
2. 打开 [shots.so](https://shots.so)
3. 将截图拖入页面
4. 选择背景样式（渐变 / 模糊 / 实色）
5. 导出 PNG

### 2. screely.com（免费）

1. 截图步骤同上
2. 打开 [screely.com](https://screely.com)
3. 上传截图，选择背景和窗口外框样式
4. 下载图片

### 3. CleanShot X（付费，macOS 原生体验最佳）

- 截图后可直接在悬浮窗口添加背景、阴影、标注
- 支持滚动截图、录屏转 GIF

### 4. ray.so（代码截图专用）

- 适合展示代码片段，不适合整个应用窗口

---

## 截图技巧

| 快捷键 | 用途 |
|--------|------|
| `Cmd + Shift + 3` | 全屏截图 |
| `Cmd + Shift + 4` | 区域截图 |
| `Cmd + Shift + 4` → `Space` → 点窗口 | 带阴影的窗口截图 ✅ 推荐 |
| `Cmd + Shift + 5` | 录屏 / 更多选项 |

## 效果建议

- 背景用**深色渐变**（如 #1a1a2e → #16213e），契合 macOS 深色风格
- 截图前将 app 调整到合适大小，留一些边距
- 一张截图只展示一个核心功能，不要截太多内容
